
# Notification
This is a notification component used for showing toast notifications accross the platform UI.
Component is used with notification reducer, notification portal and notification middleware.
But can be also used as a standalone component.

## Props

|name|type|default|description|
|----|----|-------|-----------|
|autoDismiss|<code>&#123;<br/><span className="default-prop-spacer"></span><span className="default-prop-spacer"></span><span className="default-prop-spacer"></span> "name": "boolean"<br />&#125;</code>|`true`|Flag to automatically call `onDismiss` after `dismissDelay` runs out.|
|dismissable|<code>&#123;<br/><span className="default-prop-spacer"></span><span className="default-prop-spacer"></span><span className="default-prop-spacer"></span> "name": "boolean"<br />&#125;</code>|`true`|Flag to show/hide notification close button.|
|onDismiss&#42;|(id:number&#124;string)=&gt;void||Function called after dismiss action is triggered. (id) => void|
|id&#42;|number&#124;string||Unique ID|
|variant&#42;|AlertProps['variant']||Alert variant. <a href="https://www.patternfly.org/v4/components/alert#types" target="_blank">More info.</a>|
|title&#42;|React.ReactNode||Alert title|
|description|React.ReactNode||Alert description|
|dismissDelay|`number`|`8000`|Time period after which `onDismiss` is called.|
|requestId|`string`||Unique request ID.|
|sentryId|`string`||Unique sentry error ID.|
