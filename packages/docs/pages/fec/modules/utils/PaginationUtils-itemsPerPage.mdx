# itemsPerPage

- Checks if the table shows the correct number of items per page.

## Params
|Name|Type|Default|Description|
|---|---|---|---|
|totalLength&#42;|<code>number</code>|<code></code>|the length of fixtures array e.g. the amount of items overall.
|pageSize&#42;|<code>number</code>|<code></code>|"items shown" setting of the table, default is 20.
